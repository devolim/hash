TOTALE ANONYMITÄT

KEIN KONTO ERSTELLEN.
KEINE IDENTITÄT PREISGEBEN.

Die meisten Messenger verlangen eine Telefonnummer oder eine E-Mail-Adresse. Hash verlangt nichts. Installieren, öffnen, chatten.

KEINE TELEFONNUMMER

Im Gegensatz zu Signal, WhatsApp oder Telegram fragt Hash niemals nach Ihrer Nummer.

KEINE E-MAIL-ADRESSE

Keine Verifizierung. Sie erhalten eine einzigartige und anonyme Hash ID. Das ist alles.

KEIN PASSWORT

Ihr Gerät ist Ihre Identität. Ein lokaler PIN schützt den Zugang — nichts wird bei uns gespeichert.

FUNKTIONEN

ALLES, WAS EIN MESSENGER SEIN SOLLTE.

Nachrichten, Audio- und Videoanrufe, Dateien — alles ist Ende-zu-Ende-verschlüsselt. Und alles verschwindet. Von Grund auf so konzipiert.

SELBSTLÖSCHENDE NACHRICHTEN

Jede Nachricht wird mit dem Signal Protocol verschlüsselt und nach Empfang vom Server gelöscht. Keine Kopie, kein Backup, keine Spur.

VERSCHLÜSSELTE ANRUFE

Audio- und Videoanrufe Peer-to-Peer, Ende-zu-Ende-verschlüsselt. Selbst wir können Ihre Gespräche nicht mithören.

SICHERE DATEIEN

Fotos, Videos, Dokumente — alles wird vor dem Versand mit AES-256 verschlüsselt. Dateien werden nach Empfang vom Server gelöscht.

"EINMAL ANSEHEN"-NACHRICHTEN

Senden Sie Fotos und Videos, die nach einmaligem Betrachten verschwinden. Unmöglich, sie erneut anzusehen oder aufzunehmen.

SICHERE NOTIZEN

Ein privater Bereich für Ihre persönlichen Notizen, durch Biometrie geschützt. Aufgabenlisten, Passwörter, Gedanken — alles bleibt lokal und verschlüsselt.

100% KOSTENLOS, OHNE GRENZEN

Kein Abonnement, keine Premium-Version, keine Werbung. Unbegrenzte Nachrichten, unbegrenzte Anrufe, unbegrenzte Dateien. Alles kostenlos.

VASH-MODUS.
ALLES VERSCHWINDET.

Ein weltweit einzigartiges Sicherheitssystem. Sie haben zwei PIN-Codes: einen zum normalen Öffnen der App und einen zweiten — den Vash-Code.

Wenn jemand Sie zwingt, Ihr Telefon zu entsperren, geben Sie den Vash-Code ein. Die App öffnet sich normal... aber alle Ihre Daten wurden lautlos gelöscht. Gespräche, Kontakte, Nachrichten — alles verschwunden. Ohne sichtbare Warnung.

Der Beobachter sieht eine leere App. Er denkt, Sie benutzen sie nicht. Sie wissen, dass Ihre Daten sicher sind. Unwiderruflich. Nicht erkennbar.

ZERO-KNOWLEDGE

SELBST WIR WISSEN NICHTS.

Hash verwendet das "Sealed Sender"-Protokoll: Der Server leitet Ihre Nachrichten weiter, ohne jemals zu wissen, wer sie sendet oder empfängt. Keine Metadaten. Keine Protokolle.

1

Sie senden eine Nachricht

Ihre Nachricht wird auf Ihrem Gerät mit dem Signal Protocol verschlüsselt. Nur der Empfänger besitzt den Schlüssel zum Entschlüsseln.

2

Der Server sieht nichts

Die Nachricht kommt in einem anonymen Postfach an. Der Server weiß nicht, wer sie gesendet hat, noch was sie enthält.

3

Der Empfänger erhält die Nachricht

Die Nachricht wird lokal entschlüsselt. Eine Empfangsbestätigung wird automatisch gesendet.

4

Die Nachricht verschwindet

Nach Bestätigung des Empfangs wird die Nachricht endgültig vom Server gelöscht. Keine Kopie wird aufbewahrt, nirgendwo.

SELBSTLÖSCHENDE NACHRICHTEN

NACH 24 STUNDEN EXISTIERT NICHTS MEHR.
KONZEPTBEDINGT.

Hash ist kein gewöhnlicher Messenger. Hier dauert kein Gespräch an. Nachrichten werden verschlüsselt, gesendet, empfangen — dann endgültig gelöscht.

1

VERSCHLÜSSELTER VERSAND

Ihre Nachricht wird auf Ihrem Gerät verschlüsselt, bevor sie gesendet wird. Der Server sieht nur einen unlesbaren Datenblock.

2

EMPFANG & LÖSCHUNG

Sobald der Empfänger die Nachricht erhält, wird sie vom Server gelöscht. Sofort. Ohne Backup.

3

MAXIMAL 24 STUNDEN

Wenn der Empfänger sich nicht verbindet, wird die Nachricht nach 24 Stunden automatisch zerstört. Keine Spur bleibt zurück.
