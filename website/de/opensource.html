Transparenz

VERTRAUEN SIE NIEMANDEM.
ÜBERPRÜFEN SIE SELBST.

Hash ist ein Open-Source-Projekt. Jede Codezeile — von der mobilen App bis zu den Serverfunktionen — ist öffentlich und überprüfbar.

ÜBERPRÜFBARER CODE

Jeder kann den Quellcode von Hash lesen und überprüfen, dass die App genau das tut, was sie verspricht.

KEINE HINTERTÜR

Bei geschlossenem Code ist es unmöglich zu wissen, ob es eine Backdoor gibt. Bei Hash ist alles sichtbar. Keine Geheimnisse.

GEMEINSCHAFT

Entwickler und Sicherheitsexperten aus der ganzen Welt können beitragen, verbessern und Hash stärken.

Mitwirken

WERDEN SIE TEIL DES PROJEKTS.

Entwickler, Designer, Sicherheitsexperte, Übersetzer — jeder kann seinen Beitrag leisten.

1

FORKEN SIE DAS REPOSITORY

Klonen Sie das Repository auf Ihren Computer und erstellen Sie einen Branch für Ihren Beitrag.

2

ENTWICKELN & TESTEN

Implementieren Sie Ihre Korrektur oder Funktion. Stellen Sie sicher, dass die Tests bestehen und der Code den Konventionen des Projekts entspricht.

3

ÖFFNEN SIE EINE PULL REQUEST

Reichen Sie Ihre PR mit einer klaren Beschreibung ein. Das Team wird sie prüfen und Ihnen Rückmeldung geben.

4

IHR CODE WIRD INTEGRIERT

Nach der Genehmigung wird Ihr Beitrag zusammengeführt. Sie sind nun Teil der Geschichte von Hash.

ENTDECKEN SIE DEN QUELLCODE.

Alles ist transparent. Durchstöbern Sie den Code, eröffnen Sie Issues, schlagen Sie Verbesserungen vor.

AUF GITHUB ANSEHEN
