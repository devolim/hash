Trasparenza

NON FIDARTI DI NESSUNO.
VERIFICA.

Hash è un progetto open source. Ogni riga di codice — dall'applicazione mobile alle funzioni server — è pubblica e verificabile.

CODICE VERIFICABILE

Chiunque può leggere il codice sorgente di Hash e verificare che l'applicazione faccia esattamente ciò che promette.

NESSUNA PORTA SEGRETA

Con codice chiuso, impossibile sapere se c'è una backdoor. Con Hash, tutto è visibile. Zero segreti.

COMUNITÀ

Sviluppatori ed esperti di sicurezza di tutto il mondo possono contribuire, migliorare e rafforzare Hash.

Contribuire

UNISCITI AL PROGETTO.

Sviluppatore, designer, esperto di sicurezza, traduttore — ognuno può dare il proprio contributo.

1

FORK DEL REPOSITORY

Clona il repository sulla tua macchina e crea un branch per il tuo contributo.

2

SVILUPPA E TESTA

Implementa la tua correzione o funzionalità. Assicurati che i test passino e che il codice rispetti le convenzioni del progetto.

3

APRI UNA PULL REQUEST

Invia la tua PR con una descrizione chiara. Il team la esaminerà e ti darà un feedback.

4

IL TUO CODICE VIENE INTEGRATO

Una volta approvato, il tuo contributo viene fuso. Ora fai parte della storia di Hash.

ESPLORA IL CODICE SORGENTE.

Tutto è trasparente. Sfoglia il codice, apri delle issue, proponi miglioramenti.
