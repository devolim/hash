Hash アカウントの削除

最終更新日：2026年2月

注意：Hash アカウントの削除は即座かつ不可逆的です。すべてのデータ（メッセージ、連絡先、暗号鍵）は永久に消去されます。削除後にアカウントを復旧する方法はありません。

アプリからアカウントを削除する

アカウントを削除する最も簡単な方法は、アプリ内のアカウント削除オプションを使用することです：

  1. デバイスで Hash アプリを開きます
  2. 歯車アイコンをタップして設定にアクセスします
  3. 「アカウント」セクションまでスクロールします
  4. 「アカウントを削除」をタップします
  5. 画面の指示に従って削除を確認します

何が起こりますか？確認後、デバイスとサーバーの両方からすべてのデータが即座に消去されます。アプリはインストール直後の初期状態に戻ります。

削除されるデータ

アカウント削除時に、以下のデータが永久に消去されます：

✕ Hash ID
✕ 公開暗号鍵
✕ 秘密鍵（デバイス上）
✕ デバイス情報
✕ 転送中の未読メッセージ
✕ すべての連絡先
✕ 会話履歴
✕ ローカルのファイルとメディア

代替方法：メールによる削除

アプリにアクセスできない場合

デバイスやアプリにアクセスできなくなった場合は、以下のアドレスにメールを送信してアカウントの削除を依頼できます：

hash-support@devolim.fr

メールには以下の情報を含めてください：

  • Hash ID（形式：000-000-XXX）をご存知の場合
  • アカウントを特定するためのその他の情報

48時間以内にご依頼を処理いたします。

削除の影響

アカウント削除後：

  • 連絡先はメッセージを送信できなくなります。以前のアカウントに送信されたメッセージは拒否されます
  • Hash ID は解放され、別のユーザーに再割り当てされる可能性があります
  • ローカルの会話はデバイスから消去されます
  • お客様が送信し、連絡先が既に受信したメッセージは相手のデバイスに残ります（他のユーザーのデバイスにローカルに保存されたデータは制御できません）
  • 削除後にアカウントやデータを復旧することは不可能です

お問い合わせ

アカウントの削除に関するご質問は、以下までお問い合わせください：

  • メール： hash-support@devolim.fr
  • プライバシー： hash-privacy@devolim.fr
