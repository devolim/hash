Hash へようこそ。本アプリケーションを使用することにより、お客様は本利用規約に同意したものとみなされます。よくお読みください。

1. サービスの概要

Hash は以下の企業が提供する安全なメッセージアプリケーションです：

  • 企業： DEVOLIM
  • SIREN： 948 000 757
  • 連絡先： hash-support@devolim.fr

Hash は、Signal Protocol に基づくエンドツーエンド暗号化で保護されたテキストメッセージ、音声・ビデオ通話、ファイル共有を可能にします。本サービスはユーザーの匿名性とプライバシーを保証するように設計されています。

2. 規約への同意

Hash アプリケーションをダウンロード、インストール、または使用することにより、お客様は本利用規約の全文を読み、理解し、同意したことを認めるものとします。これらの条件に同意しない場合は、本サービスを使用しないでください。

DEVOLIM は、本利用規約をいつでも変更する権利を留保します。変更は公開と同時に効力を生じます。変更の公開後もサービスの使用を継続することにより、お客様は更新された利用規約に同意したものとみなされます。

3. 最低年齢

Hash を使用するには13歳以上である必要があります。現地の法律で義務付けられている EU 加盟国では、最低年齢が16歳となる場合があります。

お住まいの国の法律上未成年である場合、Hash を使用するために法定代理人の同意を得ていることを宣言するものとします。

4. 登録とアカウント

4.1 アカウント作成

Hash アカウントの作成には個人データは一切不要です（電話番号もメールアドレスも不要）。一意の識別子（Hash ID）が自動的に割り当てられます。

4.2 1つのデバイス = 1つのアカウント

重要：Hash アカウントは1つのデバイスに紐づけられています。アカウントの復旧メカニズムは存在しません。デバイスの紛失、盗難、または機種変更の場合、お客様のデータ（メッセージ、連絡先、暗号鍵）は永久に失われます。これはセキュリティを保証するための意図的な設計上の選択です。

4.3 アカウントのセキュリティ

PIN コードとデバイスのセキュリティはお客様の責任です。Hash は PIN コードのリセットやデータへのアクセスを行うことができません。

5. 許容される使用

Hash を使用するにあたり、以下の目的でサービスを使用しないことに同意するものとします：

  • フランス法またはお客様の管轄地域の適用法に基づく違法なコンテンツの配信、保存、または送信
  • 他のユーザーへのハラスメント、脅迫、威嚇、または嫌がらせ
  • スパム、迷惑メッセージ、または無許可の商用通信の送信
  • 暴力の脅迫またはヘイトの扇動
  • 未成年者の搾取、虐待、または危険にさらすこと（児童性的虐待コンテンツ（CSAM）の配布を含む）
  • 詐欺、なりすまし、または欺瞞
  • マルウェア、ウイルス、またはあらゆる有害なコードの配布
  • 第三者の知的財産権の侵害
  • セキュリティ対策の回避または Hash のシステムへの不正アクセスの試み
  • テロリズムまたはテロ活動の資金援助

6. アカウントの停止および削除

DEVOLIM は、本利用規約に違反した場合、特に以下の場合にアカウントを停止または削除する権利を留保します：

  • 違法な目的でのサービスの使用
  • 他のユーザーから報告されたハラスメントまたは嫌がらせ
  • サービスのセキュリティを侵害する試み
  • Hash の精神およびプライバシー保護に反するあらゆる活動

サービスの暗号化された性質上、アカウントの削除はすべての関連データ（Hash ID、公開鍵、未読の転送中メッセージ）の不可逆的な消去を伴います。

Hash を介して送信するコンテンツについては、お客様が単独で責任を負います。お客様の通信が適用法および本利用規約に準拠していることを保証するものとします。

Hash は無料サービスです。アプリケーションの機能を使用するために支払いは不要です。Hash には広告やアプリ内課金は一切含まれていません。

12. 報告

本利用規約に違反する行為や不正な行動を発見した場合は、アプリ内の報告機能を使用して直接報告するか、hash-abuse@devolim.fr にメールでご連絡ください。エンドツーエンド暗号化のため、メッセージの内容にはアクセスできませんが、報告されたアカウントに対して措置を講じることができます。

12.1 児童性的虐待コンテンツ（CSAM）

DEVOLIM は CSAM に対してゼロトレランスの方針を採用しています。児童の性的搾取に関する信頼性のある通報を受けた場合、以下の措置を講じます：

  • 報告されたアカウントを直ちに停止
  • 18 U.S.C. § 2258A に基づき、National Center for Missing & Exploited Children（NCMEC）の CyberTipline を通じて報告を提出
  • 適用法で求められる場合、法執行機関のために利用可能な非コンテンツデータ（Hash ID、タイムスタンプ、報告提出時の IP アドレス）を保存
  • 有効な法的要請に基づき法執行機関と協力

CSAM の疑いを報告するには、件名を「CSAM Report」として hash-abuse@devolim.fr まで直ちにご連絡ください。
