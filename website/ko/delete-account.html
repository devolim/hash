Hash 계정 삭제

최종 업데이트: 2026년 2월

주의: Hash 계정 삭제는 즉시 이루어지며 되돌릴 수 없습니다. 모든 데이터(메시지, 연락처, 암호화 키)가 영구적으로 삭제됩니다. 삭제 후 계정을 복구할 수 있는 방법은 없습니다.

앱에서 계정 삭제

계정을 삭제하는 가장 간단한 방법은 앱 내 계정 삭제 옵션을 사용하는 것입니다:

  1. 기기에서 Hash 앱을 엽니다
  2. 톱니바퀴 아이콘을 눌러 설정으로 이동합니다
  3. "계정" 섹션까지 아래로 스크롤합니다
  4. "계정 삭제"를 누릅니다
  5. 화면의 안내에 따라 삭제를 확인합니다

삭제하면 어떻게 되나요? 확인 즉시 기기와 서버 모두에서 모든 데이터가 즉시 삭제됩니다. 앱은 처음 설치된 것처럼 초기 상태로 돌아갑니다.

삭제되는 데이터

계정 삭제 시 다음 데이터가 영구적으로 삭제됩니다:

✕ 귀하의 Hash ID
✕ 공개 암호화 키
✕ 개인 키 (기기 내)
✕ 기기 정보
✕ 읽지 않은 전송 중 메시지
✕ 모든 연락처
✕ 대화 기록
✕ 로컬 파일 및 미디어

대체 방법: 이메일로 요청

앱에 더 이상 접근할 수 없나요?

기기나 앱에 더 이상 접근할 수 없는 경우, 다음 주소로 이메일을 보내 계정 삭제를 요청할 수 있습니다:

hash-support@devolim.fr

이메일에 다음 정보를 포함해 주세요:

  • 알고 계신 경우 Hash ID (형식 000-000-XXX)
  • 계정 식별에 도움이 되는 모든 정보

48시간 이내에 요청을 처리하겠습니다.

삭제의 결과

계정 삭제 후:

  • 연락처가 더 이상 메시지를 보낼 수 없습니다. 이전 계정으로 보낸 메시지는 거부됩니다
  • Hash ID가 해제되어 다른 사용자에게 재할당될 수 있습니다
  • 로컬 대화가 기기에서 삭제됩니다
  • 이미 연락처가 수신한 메시지는 해당 연락처의 기기에 남아 있습니다 (다른 사용자의 로컬 데이터에 대해서는 당사가 통제할 수 없습니다)
  • 삭제 후 계정이나 데이터를 복구하는 것은 불가능합니다

연락처

계정 삭제에 관한 문의 사항은 다음으로 연락해 주세요:

  • 이메일: hash-support@devolim.fr
  • 개인정보 보호: hash-privacy@devolim.fr

당신에 대해 아무것도 모르는 메신저.
