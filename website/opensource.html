VÉRIFIEZ.

Hash est un projet open source. Chaque ligne de code, de l'application mobile aux fonctions serveur, est publique et vérifiable.

Code auditable

N'importe qui peut lire le code source de Hash et vérifier que l'application fait exactement ce qu'elle promet.

Pas de porte dérobée

Avec du code fermé, impossible de savoir s'il y a une backdoor. Avec Hash, tout est visible. Zéro secret.

Communauté

Développeurs et experts en sécurité du monde entier peuvent contribuer, améliorer et renforcer Hash.

Contribuer

 

Développeur, designer, expert en sécurité, traducteur : chacun peut apporter sa pierre à l'édifice.

1

Forkez le dépôt

Clonez le repository sur votre machine et créez une branche pour votre contribution.

2

Développez & testez

Implémentez votre correction ou fonctionnalité. Assurez-vous que les tests passent et que le code respecte les conventions du projet.

3

Ouvrez une Pull Request

Soumettez votre PR avec une description claire. L'équipe la reverra et vous donnera un retour.

4

Votre code est intégré

Une fois approuvée, votre contribution est fusionnée. Vous faites maintenant partie de l'histoire de Hash.
