删除您的 Hash 帐户

最后更新：2026年2月

警告：删除 Hash 帐户是即时且不可逆的操作。您的所有数据（消息、联系人、加密密钥）将被永久清除。帐户删除后无法恢复。

从应用内删除帐户

删除帐户最简单的方式是直接在应用中使用删除我的帐户选项：

  1. 在您的设备上打开 Hash 应用
  2. 点击齿轮图标进入设置
  3. 滚动到"帐户"部分
  4. 点击"删除我的帐户"
  5. 按照屏幕上的提示确认删除

删除后会发生什么？一旦您确认，您的所有数据将立即从您的设备和我们的服务器上被清除。应用将恢复到初始状态，如同刚刚安装一样。

被删除的数据

删除帐户时，以下数据将被永久清除：

✕ 您的 Hash ID
✕ 公共加密密钥
✕ 私钥（设备上的）
✕ 设备信息
✕ 传输中的未读消息
✕ 所有联系人
✕ 对话历史记录
✕ 本地文件和媒体

替代方法：通过电子邮件

无法访问应用？

如果您无法访问设备或应用，可以通过以下方式发送邮件请求删除帐户：

hash-support@devolim.fr

请在邮件中提供：

  • 您的 Hash ID（格式 000-000-XXX），如果您知道的话
  • 任何有助于识别您帐户的信息

我们将在 48小时内处理您的请求。

删除后的影响

帐户删除后：

  • 您的联系人将无法再向您发送消息。发送到您旧帐户的消息将被拒绝
  • 您的 Hash ID 将被释放，可能被分配给其他用户
  • 您的本地对话将从您的设备上被清除
  • 您之前发送的已被联系人接收的消息仍保留在他们的设备上（我们无法控制其他用户在本地存储的数据）
  • 帐户删除后无法恢复您的帐户或数据

联系方式

如对帐户删除有任何疑问，请联系我们：

  • 电子邮件：hash-support@devolim.fr
  • 隐私事务：hash-privacy@devolim.fr

对您一无所知的通讯应用。
