快速入门

从 App Store（iOS）或 Google Play（Android）免费下载 Hash。该应用永久免费，无内购，无订阅。
不需要。Hash 不索取任何手机号码、电子邮箱或密码。首次启动时，您只需设置一个本地 PIN 码，系统会自动生成一个匿名 Hash ID。就这些。
Hash ID 是分配给您设备的唯一匿名标识符。它不与任何个人信息关联。没有人可以通过您的 Hash ID 追踪到您。
有两种方法可供选择：

1. 二维码（推荐） — 其中一方在应用中显示自己的二维码，另一方扫描。连接即时且匿名。

2. 临时码 — Hash 生成一个有效期为 5 分钟的 6 位数字码。将其分享给联系人，对方在应用中输入即可添加您。
Hash 采用"一台设备 = 一个账户"的原则。这增强了您的安全性：加密密钥只存在于一台设备上。不过，您可以通过设置将账户迁移到新设备。

消息

是的。一旦接收方收到消息，它会立即从我们的服务器上删除。如果接收方在 24 小时内未上线，消息将自动且永久销毁。不保留任何副本。
阅后即焚媒体是接收方只能查看一次的照片或视频。查看后，它将永久消失，无法重新查看或保存。
您可以发送照片、视频、文档、语音消息和任何类型的文件。所有文件在发送前均使用 AES-256-GCM 加密，接收后从服务器删除。
Hash 使用 Signal 协议对您的消息进行端到端加密。每条消息在发送前都在您的设备上加密。服务器只能看到无法读取的数据块。只有您的接收方拥有解密密钥。此外，Sealed Sender 协议确保服务器也无法知道谁在向谁发送消息。
可以。Hash 允许您以加密方式分享位置。您的位置与消息使用相同的加密方式，接收后从服务器删除。

通话

是的。所有语音和视频通话均进行端到端加密。连接直接在设备之间建立（点对点）。不进行任何录音。不保留任何通话时长或通话记录。
通话质量取决于您的网络连接。为获得最佳体验：

• 使用稳定的 Wi-Fi 连接
• 确保您的 4G/5G 连接充足
• 关闭其他占用带宽的应用
• 确认双方都有良好的网络连接
可以。Hash 支持语音和视频通话，两者均进行端到端加密。您可以在通话过程中在语音和视频之间切换。

安全

Vash 模式是 Hash 独有的功能。您设置第二个 PIN 码。如果有人强迫您打开应用，您输入这个第二个 PIN 码。应用正常打开，但所有数据会被静默且不可逆地删除——联系人、消息、对话、文件。旁观者看到的是一个空应用，没有任何警告显示。

警告：此操作不可逆。所有数据将永久丢失。
恐慌按钮可通过单次操作立即永久删除应用的所有数据。与 Vash 模式不同，此删除是可见的——用户明确确认销毁数据。
摇晃锁定允许您通过摇晃手机立即锁定应用。当有人在您使用 Hash 时靠近您，此功能非常有用。您可以在设置中调整检测灵敏度。
Sealed Sender 是一种阻止服务器知道谁在与谁通信的协议。您的消息完全匿名传输：发送方、接收方和内容对我们的基础设施都是不可见的。即使有人拦截流量，也无法确定谁在向谁发送消息。
根据设计，Hash 不在我们的服务器上存储任何可用于重置 PIN 的信息。如果您设置了恢复短语，可以使用它来恢复访问权限。否则，应用数据将不可挽回地丢失，您需要重新安装 Hash 从头开始。这是一个安全选择，而非缺陷。
不能。端到端加密意味着只有您和您的接收方才能读取您的消息。服务器只能看到在技术上无法解密的加密数据。此外，Sealed Sender 协议防止服务器知道谁在与谁通信。即使我们收到法律命令，也不可能提供您对话的内容。

通知

进入设置 > 通知，修改显示名称。您可以设置"提醒"、"备忘录"或任何名称。通知将显示此名称而非"Hash"，让应用在日常生活中不引人注意。
在设置 > 通知中，启用隐藏内容选项。通知将显示通用消息而非您实际消息的内容。没有人能从您肩膀后面偷看您的对话。
请检查以下几点：

• 权限 — 确保 Hash 有发送通知的权限（手机设置 > Hash > 通知）
• 勿扰模式 — 检查是否已开启
• 电池 — 某些手机在省电模式下会禁用通知。将 Hash 排除在电池优化之外
• 连接 — 检查您的网络连接

账户

进入设置 > 账户 > 迁移我的账户。应用会生成一个迁移码，在新设备上输入此码。迁移会将您的 Hash ID 和联系人转移到新设备。旧设备将自动断开连接。

注意：消息不会被迁移，这符合 Hash 阅后即焚消息的原则。
进入设置 > 账户 > 删除我的账户。删除即时且不可逆。您的所有数据（Hash ID、联系人、加密密钥、消息）将从您的设备和我们的服务器上永久抹除。

您也可以通过电子邮件申请删除：hash-support@devolim.fr

故障排除

检查您的网络连接。连接恢复后消息会自动发送。如果问题持续存在，尝试关闭并重新打开应用。如果您的联系人已删除账户或将您拉黑，消息将无法送达。
请尝试以下步骤：

• 确保您使用的是最新版本的 Hash
• 重启您的手机
• 检查您的操作系统是否已更新
• 如果问题持续，请发送邮件至 hash-support@devolim.fr 并描述问题
发送电子邮件至 hash-support@devolim.fr，附上问题描述、您的设备型号和操作系统版本。截图或重现错误步骤的详细描述非常有帮助。
