隐私政策

最后更新：2026年2月

概要：Hash 的设计理念是几乎不收集任何个人数据。您的消息经过端到端加密，我们无法读取。没有数据分析、没有广告、不与任何第三方共享数据。

2. 收集的数据

Hash 遵循数据最小化原则设计。我们仅收集服务运行所需的最低限度数据：

2.1 我们收集的数据

  • Hash ID：随机生成的唯一标识符（格式 000-000-XXX），与您的真实身份无关
  • 设备标识符：用于将您的帐户与设备关联的唯一技术标识符
  • 平台：所使用的设备类型（iOS 或 Android），用于发送通知
  • 公共加密密钥：您的 Signal Protocol 密钥环中的公钥，供您的联系人向您发送加密消息
  • 邮箱令牌：从您的密钥加密派生的令牌，用于匿名接收消息

2.2 我们不收集的数据

  • 消息内容：所有消息均经过端到端加密（E2E）。即使我们想要读取也无法做到
  • 联系人列表：您的通讯录保留在您的设备上，我们从不访问
  • 位置信息：地理位置数据仅在您自愿发送位置消息时传输。数据经过端到端加密，我们的服务器无法读取或存储
  • 电子邮件地址：不要求提供，不予收集
  • 电话号码：不要求提供，不予收集
  • IP 地址：不存储在我们的数据库中
  • 通信元数据：借助匿名令牌系统，我们无法得知谁在与谁通信
  • 照片、文件、媒体：所有多媒体内容在发送前均在客户端加密，仅以不可读的加密数据形式经过我们的服务器

3. 加密与安全

Hash 采用尖端加密技术来保护您的通信：

  • Signal Protocol：端到端加密协议，Signal 和 WhatsApp 也在使用，提供前向保密和未来保密
  • AES-256：军事级对称加密算法，用于消息和文件加密
  • Curve25519：用于 Diffie-Hellman 密钥交换的椭圆曲线
  • Double Ratchet：每条消息使用唯一的会话密钥，自动更新
  • Sealed Sender：匿名发送者技术，防止服务器获知消息发送者

您的私钥永远不会离开您的设备。它们存储在本地加密存储空间（Hive）中，从不传输到我们的服务器。

4. 数据保留期限

  • 传输中的消息：在服务器上存储最长24小时。一旦收件人确认接收，消息将立即且不可逆转地从服务器删除
  • 帐户数据：在帐户活跃期间保留。帐户删除时（紧急按钮或手动删除）立即永久删除
  • 文件和媒体：与消息相同的策略，确认接收后即删除

5. 本地存储

本地数据（解密的消息、联系人、密钥）仅存储在您的设备上，存储于加密的 Hive 数据库中。这些数据从不同步到我们的服务器或云端。

如果您丢失设备或更换设备，这些数据将永久丢失。这是有意的设计选择：一台设备 = 一个帐户，不提供恢复功能。

6. 与第三方共享数据

不共享任何数据。Hash 不使用任何分析工具、广告或跟踪器。我们不会与任何第三方共享、出售或转让任何数据。

我们的服务器由 Supabase 托管。存储在这些服务器上的数据要么是不可读的加密数据（消息），要么是最基本的技术数据（Hash ID、公钥）。

推送通知通过 Firebase Cloud Messaging（Google LLC）发送。Google 仅处理 FCM 设备令牌用于此目的。不通过此服务传输任何消息内容。Firebase 隐私政策。

7. 您的权利（GDPR）

根据《通用数据保护条例》（GDPR），您享有以下权利：

  • 访问权：您可以请求了解我们持有的关于您的数据（Hash ID、公钥、设备标识符）
  • 更正权：您可以要求更正不准确的数据
  • 删除权：您可以随时通过应用内的紧急按钮或设置部分删除所有数据。删除即时且不可逆
  • 数据可携带权：鉴于收集的数据量极少，您可以联系我们获取您的数据副本
  • 反对权：您可以通过删除帐户来反对数据处理
  • 投诉权：您可以向 CNIL（www.cnil.fr）提交投诉

如需行使您的权利，请联系我们：hash-privacy@devolim.fr。

8. 您的权利（美国隐私法）

如果您是加利福尼亚州（CCPA/CPRA）、弗吉尼亚州（VCDPA）、科罗拉多州（CPA）或其他拥有隐私立法的美国州的居民，您享有以下权利：

  • 知情权：您可以请求了解我们收集的个人数据及其使用方式
  • 删除权：您可以随时通过应用内删除功能或联系我们来请求删除您的个人数据
  • 拒绝出售的权利：我们不会以金钱或其他有价对价的方式向第三方出售、共享或出租您的个人数据
  • 不受歧视的权利：我们不会因为您行使隐私权利而歧视您

如需行使这些权利，请使用应用内帐户删除功能或联系我们：hash-privacy@devolim.fr。

9. 最低年龄

Hash 面向13周岁或以上的用户。在某些欧盟成员国，根据当地法律，最低年龄可能为16周岁。

我们不会故意收集低于适用最低年龄的儿童数据。如果您是父母或监护人，且认为您的孩子在未经您同意的情况下使用了 Hash，请联系我们：hash-privacy@devolim.fr。

10. Cookie 和跟踪技术

Hash 应用不使用任何 Cookie、跟踪像素或分析 SDK。hash.devolim.fr 网站同样不使用跟踪 Cookie。

11. 国际数据传输

最基本的技术数据（Hash ID、公钥）可能在欧洲经济区以外的服务器上处理。然而，鉴于所有敏感数据均经过端到端加密，且我们不收集任何可识别的个人数据，因此对您的隐私风险极低。

12. 数据安全

我们实施以下安全措施：

  • 所有通信端到端加密（Signal Protocol）
  • 本地数据 AES-256 加密
  • PIN 码和生物识别认证
  • 胁迫密码（duress PIN）用于紧急擦除
  • 防截屏保护（Android 上的 FLAG_SECURE）
  • 消息送达后自动从服务器删除
  • 仅通过安全函数（security definer）访问数据库
  • 所有表均设有 RLS（行级安全）策略

13. 政策变更

我们可能会不定期更新本隐私政策。如有重大变更，我们将通过应用内通知告知您。最后更新日期标注在本页顶部。
